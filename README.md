# LibraryProgram

## 주제
도서 관리 프로그램

## 요구사항
책의 제목, 저자, 장르 등을 입력받는다.  
입력받은 책의 정보(제목, 저자, 장르 등)를 확인할 수 있다.  
선택한 책을 대여할 수 있다.  

## 실행내용
===============
1. 전제 도서 목록
2. 도서 번호로 조회
3. 도서 제목으로 조회
4. 도서 등록
5. 도서 삭제
6. 도서 대여
0. 프로그램 종료
